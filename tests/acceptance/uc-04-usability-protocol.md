# UC-04 Usability Protocol (SC-005)

## Objective
Measure first-attempt completion success for author paper submission.

## Success Criterion
- SC-005 passes when >= 90% of participants complete UC-04 on first attempt without external assistance.

## Procedure
1. Provide participant with authenticated author context.
2. Ask participant to submit one valid paper (metadata + manuscript file) in one attempt.
3. Record whether submission reaches `submitted` state without facilitator intervention.
4. Record blockers and correction categories when first attempt fails.

## Data Capture
- Participant ID
- First-attempt result (`success`/`failure`)
- Time-to-complete (seconds)
- Notes
